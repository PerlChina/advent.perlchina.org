如果上述还不够过瘾，试试下面的。

编辑一个 hello.pl

=begin code

print "Hello World\n";

=end code

然后创建一个 havefun.pl, paste 如下代码：

=begin code

use Acme::EyeDrops qw(sightly);
print sightly( {
    Shape       => 'camel',
    SourceFile  => 'hello.pl',
    Regex       => 1
} );

=end code
